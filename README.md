# eslint-config

ESLint config for how I prefer javascript code.

## Usage

    Copy to root of project and rename to .eslintrc

## Usage

Run `eslint` on  javascript files:

    eslint **/*.js

## Configurations

### .eslintrc-es5

Linting rules for ES5.
